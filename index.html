  • Un simbolo se crea apartir de la de la adición de dos puntos(:) delante de una palabra.

def contains_method name
  self.methods.each do |method|
    incoming_method = name.to_s
    incoming_method += '!'
    if method.to_s.eql?  incoming_method
      return true
    end
  end
  false
end

Implementando FancyMarkup


class FancyMarkup
  attr_accessor :indents, :html
  def initialize
    @indents = 0
    @html = ''
  end
  def method_missing(m, *args, &block
   tag(m, args, &block)
  end
  def document(*args, &block)
    tag(:html, args, &block)
  end
  #..
end
private
    def tag(html_tag, args, &block)
      content = find_content(args)
      options = html_options(find_options(args))
      html << "\n#{indent}<#{html_tag}#{options}>#{content}"
      if block_given?
        @indents += 1
        instance_eval(&block)
        @indents -= 1
        html << "\n#{indent}"
      end
      html << "<{html_tag}>"
    end
                                def find_content(args)
  args.detect{|arg| arg.is_a? String}
end
def find_options(args)
  args.detect{|arg| arg.is_a? Hash} || {}
end
def indent
  ' ' * indents
end
def html_options(options)
  options.collect{ |key, value|
    value = value.to_s.gsub('"', '\"')
    " #{key}=\"#{value}\""
  }.join('')
end

Ruby on Rails

Que es Rails?

“Ruby on Rails es un web application framework escrito en Ruby. Utiliza el patrón de diseño Modelo-Vista-Controlador(MVC) para organizar la programación de la aplicacion.”

Arquitectura de RoR

Arquitectura en RoR

  • Un Modelo Mapea a una tabla en una Base de Datos

  • Un Controlador es el componente que responde a peticiones externas del webserver a la aplicación, y responde a dicha petición externa determinando cual vista renderear.

  • Una vista es, en la configuración por defecto de Rails un archivo erb. Este es típicamente convertido a html en tiempo de ejecución.

¿Qué es tan especial acerca de Rails?

  • Otros frameworks usan generación extensiva de código, que le da al usuario un adelanto de productividad, y scripts de customización le permiten customizar dicho código en un pequeño número de puntos seleccionados cuidadosamente.
    • Metaprogramación reemplaza estas dos técnicas primitivas y elimina sus desventajas
    • Ruby es uno de los mejores lenguajes para metaprogramación, y Rails hace uso extensivo de dicha elemento.

  • El código de configuración se reduce a casi una quinta parte del código de configuración de otros framework solo por seguir configuraciones comunes.
    • Nombrando las clases que representan tus modelos con el mismo nombre de la correspondiente tabla en la Base de Datos.
    • 'id' es el nombre de la llave primaria.

Active Record

  • Salva objetos a la Base de Datos
  • Descubre la columna en el esquema de la Base de Datos y automáticamente lo incluye al dominio del objeto usando metaprogramación.

Action Pack

  • Vista y Controlador tienen una alta interacción, en Rails ellos se combinan en Action Pack.
  • Action Pack divide una peticion web en componentes de la vista y componentes del controlador.
